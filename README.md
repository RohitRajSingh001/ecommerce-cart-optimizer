# ⚡ Cartify — Smart E-Commerce Cart Optimizer & Budget Assistant

Cartify is an industry-grade, responsive full-stack shopping assistant designed to optimize consumer purchasing power under strict budget limits. Built with **React 19**, **Nginx**, **Node.js/Express**, and fully containerized with **Docker Compose**.

## 🌟 Key Features
- 🧠 **Greedy Budget Optimizer**: A client-side knapsack algorithm that dynamically selects the highest-value, highly discounted items in your cart to perfectly fit within your target budget.
- 🎨 **10/10 Premium Glassmorphic UI**: Responsive design featuring slate-toned Light & Dark modes, fluid CSS animations, and a visual budget utilization meter.
- 🐳 **Docker & Nginx Orchestration**: Fully containerized multi-service setup featuring multi-stage production builds served over an ultra-light Nginx web server.
- 🔔 **Interactive UI Systems**: Real-time catalog searching, dynamic category filtering, a slide-over cart drawer, async toast alerts, and a simulated checkout success modal.
